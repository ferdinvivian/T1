Readme, FAQ, Errata...

This is a short downloadable ‘readme’ file and FAQ that introduces the course and provides answers to some interesting questions that have been asked by students. Any corrections or clarifications are also listed in an errata. Please download and read this document before asking any questions!

Questions...

If you wish to ask a question about a specific course topic, please post it online in the Q&A thread next to the video lecture. Please do not send me private messages to ask questions. Your question may be useful to other students so I like to make sure that all questions and answers appear in the public forums.
